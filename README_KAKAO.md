# 🗺️ 카카오 맵 연동 가이드 (Kakao Map Setup Guide)

카페드림의 상권 분석 지도가 정상적으로 작동하려면 아래 **3가지 단계**를 완료해야 합니다.

---

### 1단계: 카카오 개발자 센터에서 키 발급
1. [카카오 개발자 콘솔](https://developers.kakao.com/)에 접속합니다.
2. [내 애플리케이션] -> [애플리케이션 추가하기] (이름: 카페드림)
3. 생성된 앱의 **[플랫폼]** 메뉴로 이동합니다.
4. **[Web 플랫폼 등록]**을 클릭하고 아래 도메인들을 추가합니다:
   - `http://localhost:3000` (로컬 테스트용)
   - `https://coffee-marketing.vercel.app` (버셀 배포 주소)
5. **[앱 키]** 메뉴에서 **JavaScript 키**를 복사합니다.

---

### 2단계: 환경 변수 설정 (로컬 & 버셀)

#### A. 로컬 컴퓨터 (`.env.local`)
`.env.local` 파일을 열고 복사한 키를 넣습니다:
```env
NEXT_PUBLIC_KAKAO_MAP_KEY=여기에_발급받은_자바스크립트_키_입력
```

#### B. 버셀 배포 환경 (Vercel Dashboard)
1. Vercel 프로젝트 설정의 **Environment Variables**로 이동합니다.
2. Name: `NEXT_PUBLIC_KAKAO_MAP_KEY`
3. Value: `복사한_자바스크립트_키`
4. 등록 후 **재배포(Redeploy)**를 진행해야 실제 사이트에 반영됩니다.

---

### 3단계: 확인 방법
1. 브라우저에서 사이트 접속 후 **대시보드** 섹션으로 이동합니다.
2. 지도가 검은색 화면 대신 실제 지도로 나타나면 성공입니다!
3. 주소 검색창에 근처 카페 주소를 입력하여 **상권 분석** 기능을 테스트해 보세요.

---

> [!IMPORTANT]
> **"지도 API 인증에 실패했습니다"**라는 메시지가 뜬다면?
> - 카카오 플랫폼 설정에 등록된 도메인과 현재 접속한 주소가 일치하는지 다시 한번 확인해 주세요.
